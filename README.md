# 生日小记

一个帮你记住身边人生日的微信小程序：记录生日、提前提醒，支持**公历 / 农历**两种生日，给每个人维护礼物账本和心愿单，用 **AI 生成专属生日祝福语**，还能通过云开发让好友自己填写生日信息。

## 功能

### 生日管理
- 添加 / 编辑 / 删除生日，记录姓名、关系、生日、联系方式、备注
- **公历 / 农历**两种生日类型，农历自动换算下一次对应的公历日期
- 首页统计：总人数、今天过生日、7 天内、本月（农历生日按换算后的公历日期统计）
- 今日生日横幅提醒、按临近程度排序的生日列表
- 提前提醒天数可自定义

### 礼物账本 + 心愿单
- 每个人的详情页可切换「基本信息 / 礼物 · 心愿」
- 礼物账本：记录历年送过的礼物，避免重复
- 心愿单：记录对方想要的东西，送礼有参考

### AI 祝福语
- 详情页一键「生成 AI 祝福语」，根据姓名、关系、生日、备注自动生成个性化短祝福
- 生成结果可一键复制
- 由 MiniMax 大模型生成（调用 `api.minimaxi.com`）

### 社交邀请 + 云端收件箱
- 生成专属邀请，好友打开后自己填写生日信息（同样支持公历 / 农历）
- 提交的信息进入云端收件箱，确认后一键导入生日簿
- 收件箱支持导入 / 忽略
- 每位用户有独立的 `invite_token`（存于本地，用于关联收到的邀请）

## 技术栈

- 微信小程序原生开发（WXML / WXSS / JS）
- 本地数据：`wx.Storage`
- 农历换算：内置 `utils/lunar.js`（查表法，覆盖 1900–2100 年）
- 云端：微信云开发（云数据库 + 云函数）
- AI：MiniMax 文本生成接口

## 项目结构

```
miniprogram/
  app.js / app.json / app.wxss      # 全局配置与样式
  pages/
    index/    首页（统计、今日/近期生日、好友邀请入口）
    add/      添加 / 编辑生日（支持公历、农历）
    list/     生日簿列表
    detail/   详情（基本信息 / 礼物 · 心愿 tab，AI 祝福语）
    invite/   好友填写生日的页面
    inbox/    云端收件箱
  utils/
    birthday-service.js             # 生日数据读写（含礼物/心愿），单一数据源
    date.js                         # 日期与统计工具
    lunar.js                        # 农历换算
cloudfunctions/
  submitInvitation/                 # 好友提交生日信息
  getInboxItems/                    # 拉取当前用户的收件箱
  markImported/                     # 标记已导入/忽略
  generateQRCode/                   # 生成邀请小程序码（需发布后可用）
```

## 运行

1. 用微信开发者工具打开本项目
2. 在 `project.config.json` 中将 `appid` 改为你自己的小程序 AppID
3. 开通微信云开发，在 `miniprogram/app.js` 中把 `env` 改为你的云开发环境 ID
4. 右键 `cloudfunctions/` 下各云函数，逐个「上传并部署」
5. （可选）启用 AI 祝福语：在 `miniprogram/pages/detail/detail.js` 顶部的 `token` 填入你自己的 MiniMax API Key；本地调试需在开发者工具「详情 → 本地设置」勾选「不校验合法域名」，正式发布则需在小程序后台把 `https://api.minimaxi.com` 加入 request 合法域名
6. 编译运行

> ⚠️ 不要把真实的 API Key 提交到公开仓库。本仓库中 `detail.js` 的 `token` 默认为空字符串。

## 说明

- 个人主体小程序无法使用分享，且未发布时无法生成小程序码。`generateQRCode` 在小程序正式发布后会自动生效；未发布时邀请弹窗进入「演示模式」，可直接打开填写页演示完整流程。
- 给他人体验需在小程序后台「成员管理 → 体验成员」添加对方微信号，并上传设为体验版。
- 农历换算为查表实现，精度约 ±2 天，覆盖 1900–2100 年。
